Avoir un server compatible PHP

Copier le dossier 'intervention' à la racine du server (Il n'est pas possible de le renomer à moins de mettre a jours les différent script!)

Mettre à la racine du server (sur la page index.html ou son script) le code suivant pour géré la redirection automatique des QR code vers la bonne page:

window.onload = function() {
    // Récupérer l'URL actuelle
    const currentUrl = new URL(window.location.href);
    
    // Extraire le paramètre 'inter' de l'URL
    const params = new URLSearchParams(currentUrl.search);
    const interLieu = params.get('inter');
    
    // Logique pour rediriger l'utilisateur vers la page correcte
if (interLieu) {
        // Si 'inter' est présent mais pas AUTOUSER, rediriger vers la page avec 'inter'
        window.location.href = `intervention/?&inter=${interLieu}`;
    } 

};
